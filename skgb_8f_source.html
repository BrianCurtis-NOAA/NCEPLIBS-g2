7C> This subroutine searches a file for the next grib 1 message. A grib
8C> 1 message is identified by its indicator section, i.e. an 8-byte
9C> sequence with 'grib' in bytes 1-4 and 1 in byte 8. If found, the
10C> length of the message is decoded from bytes 5-7. The search is
11C> done over a given section of the file. The search is terminated if
12C> an eof or i/o error is encountered.
13C> PROGRAM HISTORY LOG:

18C> messages.
19C> - 2009-12-14 Boi Vuong modified to increase length of seek (512).
20C>

22C> @param[in] ISEEK integer number of bytes to skip before search.
